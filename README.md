## 🏛️ Tentang PUSPA

**PUSPA (Pertubuhan Urus Peduli Asnaf)** ialah pertubuhan NGO berdaftar yang berkhidmat untuk komuniti asnaf di **Hulu Klang, Gombak, Ampang** (Selangor/KL, Malaysia).

| Butiran | Maklumat |
|---|---|
| **Pendaftaran** | PPM-006-14-14032020 |
| **Pengecualian Cukai** | LHDN s44(6) |
| **Lokasi** | Hulu Klang, Gombak, Ampang |

PUSPA V4 ialah platform pengurusan NGO generasi keempat yang mengintegrasikan **kecerdasan buatan (AI)** untuk mengautomasi operasi harian, mempercepat proses kes, dan memastikan pematuhan regulasi — semua dalam satu papan pemuka bersepadu.

## 🤖 Hermes — Pembantu AI

Hermes ialah pembantu AI autonomi yang diilhamkan oleh senibina **NousResearch Hermes Agent**. Ia belajar dari setiap interaksi dan bertambah baik dari masa ke masa.

### Aliran Kerja

```
Input Pengguna → Hermes Store → /api/v1/hermes/chat
  → Bina Promp Sistem (konteks + memori + kemahiran)
  → Panggil LLM (Z-AI / OpenRouter / Ollama)
  → Huraikan tag <<TOOL:name>>{}<</TOOL>> atau panggilan fungsi natif
  → Laksanakan rantai alat (sehingga 5 langkah)
  → Format respons dengan hasil alat
  → Ekstrak & simpan memori (tidak menyekat)
  → Simpan perbualan ke DB (tidak menyekat)
  → Kembalikan respons (dengan penstriman SSE pilihan)
```

### Sokongan Multi-Pembekal

| Pembekal | Jenis | Kos | Keterangan |
|---|---|---|---|
| **Z-AI SDK** | Awan | Percuma | Lalai, tiada konfigurasi diperlukan |
| **OpenRouter** | Awan | Percuma + Berbayar | 200+ model, panggilan fungsi natif |
| **Ollama** | Tempatan | Percuma | Privasi terlebih dahulu, inferens tempatan |

### Daftar Alat (30+)

**Alat Pertanyaan** (baca sahaja):
- `query_stats` — Statistik agregat untuk mana-mana modul
- `search_members`, `search_cases`, `search_donors`, `search_volunteers`
- `get_member_details`, `get_case_details`
- `get_donations_summary`, `list_programmes`
- `compliance_status`, `get_dashboard_analytics`

**Alat CRUD** (baca + tulis):
- `create_member`, `update_member`
- `create_case`, `update_case_status`, `add_case_note`
- `create_donation`, `create_disbursement`

**Alat Analitik**:
- `analyze_trends`, `generate_report`, `risk_assessment`

**Alat Sistem**:
- `manage_memory`, `manage_skill`, `search_documents`
- `navigate_to`, `create_record`, `update_record`

### Sistem Kemahiran

Kemahiran yang bertambah baik sendiri dalam format **SKILL.md**:
- 12 kemahiran lalai yang disemai merangkumi aliran kerja utama
- Kemahiran dipelajari dari interaksi dan bertambah baik dari masa ke masa
- Menjejaki `usageCount` dan `successRate`
- Penciptaan automatik dari interaksi kompleks
- Dimasukkan ke dalam promp sistem dengan arahan terperinci apabila dicetuskan

### Sistem Memori

Memori berterusan dengan 5 kategori:

| Kategori | Keterangan |
|---|---|
| **preference** | Keutamaan pengguna (bahasa, format, tahap butiran) |
| **fact** | Fakta pengguna (nama, organisasi, cawangan) |
| **procedure** | Aliran kerja yang dipelajari |
| **relationship** | Hubungan profesional |
| **context** | Konteks situasional |

Ciri: pengekstrakan automatik, skor keyakinan, penjejakan akses, pengingatan berasaskan kaitan.

# Tetapkan pembolehubah persekitaran
DATABASE_PROVIDER=postgresql

bun run db:push

Fail skema berasingan tersedia:

## 🔒 Ciri-Ciri Keselamatan

| Ciri | Keterangan |
|---|---|
| **JWT NextAuth** | Token sesi dengan tuntutan berasaskan peranan |
| **Token HMAC** | Token sesi kustom yang ditandatangani dengan HMAC |
| **Kekunci API Bot** | Kekunci perkhidmatan `psbot_` dengan penghasilan SHA-256 |
| **Pembatas Kadar** | Pembatas kadar dalam ingatan berasaskan IP (100 req/min) |
| **Log Audit** | Jejak audit penuh untuk semua tindakan penting |
| **Keselamatan Peringkat Baris** | Pembantu kawalan akses peringkat data |
| **eKYC** | Pengesahan identiti dengan OCR & pemadanan wajah |
| **TapSecure** | Kebenaran biometrik & pengikatan peranti |
| **Hash Kata Laluan** | Hashing scrypt crypto (bukan bcrypt) |
| **Muat Naik Selamat** | Senarai putih MIME, had saiz, perlindungan traversal laluan |
| **Proxy Dipercayai** | Sokongan IP proksi dipercayai untuk pembatas kadar |

## ⚙️ Pembolehubah Persekitaran

| Pembolehubah | Diperlukan | Keterangan |
|---|---|---|
| `DATABASE_URL` | ✅ | URL sambungan pangkalan data utama |
| `NEXTAUTH_SECRET` | ✅ | Rahsia penandatanganan JWT (wajib dalam pengeluaran) |
| `NEXTAUTH_URL` | ⚠️ | URL asas untuk NextAuth (auto-dikesan di Vercel) |
| `POSTGRES_PRISMA_URL` | ❌ | URL PostgreSQL alternatif |
| `SUPABASE_DB_URL` | ❌ | URL pangkalan data Supabase |
| `PUSPA_SESSION_SECRET` | ❌ | Rahsia token sesi kustom |
| `API_SECRET_KEY` | ❌ | Kunci rahsia fallback sesi/operator |
| `PUSPA_OPERATOR_PASSWORD` | ❌ | Kata laluan log masuk operator |
| `PUSPA_OPERATOR_ROLE` | ❌ | Peranan operator lalai |
| `TRUSTED_PROXY_IPS` | ❌ | IP proksi dipercayai (dipisahkan koma) |
| `DATABASE_PROVIDER` | ❌ | Tetapkan `postgresql` untuk tukar skema |

Contoh `.env`:

```env
# Pangkalan Data
DATABASE_URL=file:./db/custom.db

# Pengesahan
NEXTAUTH_SECRET=your-super-secret-key-change-this
NEXTAUTH_URL=http://localhost:3000

# Pembekal AI (pilihan)
OPENROUTER_API_KEY=your-openrouter-key

# Pangkalan Data PostgreSQL (pilihan)
# DATABASE_PROVIDER=postgresql
# POSTGRES_PRISMA_URL=postgresql://user:pass@host:5432/puspa
```
