# PUSPA-V4 App Map

Core domains:

- dashboard
- members
- cases
- donations/donors
- disbursements
- eKYC
- reports/audit
- volunteers
- programmes
- compliance
- documents
- OpenClaw/Ops Conductor

New business endpoints should live under `src/app/api/v1/*`. Internal screens usually belong in `src/modules/*`, not new top-level routes.
